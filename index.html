Seeking Interaction

D3 Interaction Graph

  • The visualization is created with d3.
  • Uses .sort to turn the data into a d3 hierarchy.
  • Uses .on("mouseover", function(){}) to fade all but the current sequence.
  • Uses .transition(), .duration(1000), and .style("opacity", 1) to change each segment to full opacity.
  • Uses group to pair each "rect" and its corresponding "text" in the legend part.
  • Uses a D3 sunburst visualization (partition layout) with data that describes sequences of events.
    (D3 Sunburst is not coverd in chapter 9 and chapter 10.)
